Class Experiences

Linux Course Project: Shell Scripting for User Management

In my Linux course at Old Dominion University, one of the most impactful assignments was creating a shell script to automate user management tasks. The course focused on essential system administration skills including working with the command line, managing users and groups, setting file permissions, and performing basic network configurations. For this assignment, I was required to write a script that could add new users, assign them to specific groups, configure directory permissions, and include error-checking for invalid input. This project simulated real-world system administration responsibilities and helped me understand how automation improves efficiency and consistency in managing Linux-based systems. Through this project, I developed practical skills in shell scripting, system automation, and command-line operations. I also improved my ability to document technical work clearly through script comments and usage instructions. Most importantly, I practiced applying cybersecurity principles such as least privilege and secure system defaults by carefully managing user permissions and error handling. These skills are crucial for roles in cybersecurity, where preventing misconfigurations and reducing manual error are key to maintaining secure environments. This assignment also reinforced my attention to detail and critical thinking, which are essential values in both cybersecurity and system administration.
